# AI_Design_Company
This is the main repo for the prototype of a AI powered customizable website where users can order Phone cases, T-Shirts, Pants, Key Chains and many more... which they can design them selves using AI.


# Competitor Research

## Printify
✅ Strengths: API, huge supplier base  
❌ Weaknesses: No AI integration

## Redbubble
✅ Great for artists  
❌ Weaknesses: Not personalized

## Fotor AI / Canva Print
✅ Simple customization  
❌ Limited creative freedom

## Opportunity:
Combine AI-powered creativity with real-world printing.
